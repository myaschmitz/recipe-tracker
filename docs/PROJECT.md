# Recipe Tracker

A modern web application for managing and organizing your favorite recipes with recipe creation, tagging, collections, and easy search functionality.

## Overview

Recipe Tracker is a full-stack web application built with Next.js that allows users to create, manage, and organize their recipes. The application features a clean, intuitive interface for adding recipes with ingredients, instructions, and tags, as well as organizing them into collections for easy discovery.

## Features

### Core Functionality
- **Recipe Management**: Create, edit, and delete recipes with detailed information
- **Ingredient Tracking**: Add and manage ingredients for each recipe
- **Step-by-Step Instructions**: Clear, organized cooking instructions
- **Recipe Tags**: Categorize recipes with custom tags (e.g., "Italian", "Quick", "Vegetarian")
- **Collections**: Organize recipes into themed collections
- **Search & Filter**: Find recipes by name, tags, or ingredients
- **Responsive Design**: Works seamlessly on desktop and mobile devices

### Advanced Recipe Features
- **Recipe Scaling**: Dynamically adjust ingredient quantities for different serving sizes
- **Fraction Calculations**: Automatic conversion of scaled amounts to proper fractions
- **PDF Export**: Export recipes to PDF format for printing or sharing
- **Recipe Favoriting**: Save and organize favorite recipes from other users
- **Want-to-Make List**: Bookmark recipes you plan to try later
- **Recipe Albums**: Create themed collections and albums of recipes

### User Management
- **User Accounts**: Personal recipe collections with authentication
- **Account Settings**: Update email, password, and profile information
- **User Preferences**: Customize units (imperial vs metric) and display options
- **Dashboard**: Personalized view with user-created categories and recent activity

### Data Management
- **Recipe Table View**: Comprehensive list of all recipes with filtering, sorting, and search
- **Advanced Filtering**: Filter by tags, difficulty, cook time, dietary restrictions
- **Bulk Operations**: Select and manage multiple recipes at once
- **Import/Export**: Backup and restore recipe collections

### User Interface
- Modern, clean design using Tailwind CSS
- Intuitive navigation between recipes, collections, and creation forms
- Recipe cards with visual previews and tag badges
- Form validation and user feedback with toast notifications
- Loading states and error handling

## Technology Stack

### Frontend
- **Next.js 15**: React framework with App Router
- **TypeScript**: Type-safe JavaScript development
- **Tailwind CSS**: Utility-first CSS framework
- **Shadcn/ui**: Modern React component library
- **Lucide React**: Icon library

### Backend
- **Next.js API Routes**: Server-side API endpoints
- **Supabase**: PostgreSQL database and authentication
- **TypeScript**: Full-stack type safety

### Development Tools
- **ESLint**: Code linting
- **Prettier**: Code formatting
- **Git**: Version control
